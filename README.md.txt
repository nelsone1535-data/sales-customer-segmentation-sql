# Sales Performance & Customer Segmentation Analysis

## Project Overview

This project analyzes retail sales data using SQL to understand product performance, regional trends, profitability, and customer segmentation.

The analysis simulates real-world business questions that a data analyst might investigate to support decision-making in a retail organization.

---

## Tools Used

- PostgreSQL
- SQL
- Data Analysis

---

## Dataset

This project uses a retail sales dataset commonly known as the "Superstore Dataset."

The dataset includes:

- shipping methods
- customer segments
- geographic regions
- product categories
- sales revenue
- discounts
- profit

---

## Business Questions Answered

1. Which product categories generate the most revenue?
2. Which sub-categories are the most profitable?
3. Which regions generate the highest sales?
4. Do higher discounts reduce profit?
5. Which customer segments generate the most profit?

---

## Key Insights

- Technology products generate the highest revenue.
- Certain sub-categories drive a disproportionate amount of total profit.
- Some regions significantly outperform others in sales.
- Higher discount levels are associated with lower average profit.
- Consumer customers generate the largest share of total revenue.
